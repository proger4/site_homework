# Чеклист запуска Website Builder на сервере

## Цель

Запустить Site.pro Website Builder на server и предоставить demo builder link.

## Шаги

- [ ] Зарегистрироваться в Site.pro по Revenue Share program.
- [ ] Подготовить VPS.
- [ ] Установить control panel, например cPanel или другую supported panel.
- [ ] Установить Site.pro plugin для выбранной control panel.
- [ ] Создать demo hosting account/domain.
- [ ] Открыть builder из panel.
- [ ] Проверить website creation flow.
- [ ] Предоставить demo builder link.
