# UI review: ошибки дизайна title page

## Чеклист проверки

- Согласованность company style: logo, color, typography, button style.
- Visual hierarchy: один главный headline, понятный CTA, корректные spacing.
- Mobile readability: длина headline, видимость button, crop hero image.
- Contrast: text/background и CTA/background.
- Alignment: grid, cards, margins, icon sizes.

## Возможные ошибки для описания

1. CTA button style не совпадает с company style draft.
2. В hero section слабая hierarchy: headline, subheadline и visual конкурируют за внимание.
