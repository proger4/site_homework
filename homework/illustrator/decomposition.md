# Спецификация задачи Figma и portfolio

## Задача exact redraw

Требования:

- Frame size: 535 × 330.
- Font: Jost.
- Использовать layouts и components, а не flat screenshot.
- Воссоздать spacing, typography, colors, controls и hierarchy.
- Называть layers понятно.
- Использовать reusable button, card, icon и section components.

Артефакты сдачи:

- Figma file link.
- Screenshot original vs recreated frame.
- Короткая заметка с объяснением component structure.

## Задача portfolio website

Структура Site.pro portfolio website:

1. Home/title section.
2. Works section с 2-3 works.
3. About/process section.
4. Contact CTA section.

Каждая work должна включать:

- Title.
- Short description.
- Role.
- Tools.
- Результат.
